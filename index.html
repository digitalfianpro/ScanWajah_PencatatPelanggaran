Panduan Lengkap Penggunaan Aplikasi SiPPS

Ikuti langkah-langkah di bawah ini untuk memulai pencatatan siswa berbasis wajah.

Langkah 01

Database & Excel Import

Masuk ke menu Daftar Siswa, unduh template excel standar. Isi kolom NISN, Nama Lengkap, dan Kelas, lalu klik Upload Data Excel untuk menyelaraskannya ke server Cloud Firestore Anda secara otomatis.

Langkah 02

Perekaman Wajah (Biometrik)

Klik menu Perekaman Wajah, isi NISN siswa yang bersangkutan. Posisikan wajah di depan kamera bawaan HP/Laptop secara pas. Klik tombol Kunci Foto Wajah untuk menangkap profil biometrik mereka, kemudian klik simpan.

Langkah 03

Scan & Tindak Lanjut BK

Akses menu Scan Pelanggaran. Hadapkan siswa ke kamera, sistem pemindai otomatis bawaan akan mendeteksi dan memvalidasi identitas siswa dalam database. Setelah terverifikasi, tentukan jenis Tindak Lanjut Guru BK pada panel di sebelah kanan.

Langkah 04

Pemantauan & Evaluasi Sanksi

Di menu Dashboard, amati akumulasi statistik pelanggaran secara berkala. Seluruh rekor penanganan bimbingan siswa akan dihitung otomatis secara kumulatif untuk memudahkan Bimbingan Konseling (BK) melakukan pemantauan berkala.

Pertanyaan Sering Diajukan (FAQ)

Tanya: Mengapa aplikasi tidak meminta download model AI lagi?

Jawab: Aplikasi ini telah diperbarui menggunakan modul pemindaian native HP/Laptop yang super cepat dan kompatibel langsung. Hal ini menghilangkan waktu loading berat download model AI sebelumnya.

Tanya: Apakah data saya tersimpan permanen?

Jawab: Ya, karena aplikasi ini telah terintegrasi dengan Firebase Cloud Firestore Anda secara langsung. Seluruh perubahan data langsung disinkronkan ke cloud secara real-time.
